Общие сведения

Программная система "Управление кофигурацией" позволяет удобно работать с документооборотом компании или в личных целях.
Приложение можно использовать как локально, так и удаленно через браузер, разместив его на хостинге.

Введение

Данное справочное руководство предлагает описания сценариев работы программной системы "Управление конфигурацией".
Разработчик: студент группы 21-ВТ-2 Нейенбург Т.А.

Вход в систему

Для входа в систему необходимо ввести имя пользователя и пароль (данные для входа в систему выдаются администратором).

После входа в панели навигации отобразится роль аккаунта (пользователь или администратор)

Просмотр даннных в пользовательском режиме

После входа на странице отображается таблица с даннными по документам, авторам документов и отделами разработчиков.

Для переключения между отображаемыми таблицами необходимо нажать на соответствующее название таблицы в панели над ней.

После чего таблица обновится

Для таблицы документов доступна выборка данных по параметрам: документ, автор, отдел.
Для формирования запроса выберете тип параметра и его значение в правой панели над таблицей.

После нажатия на кнопку выборки таблица обновится

Просмотр и редактирование даннных в режиме администратора

Просмотр данных в режиме администратора не отличается от просмотра в режиме пользователя.
Для удаления/добавления/обновления данных в панели администратора добавляются следующие элементы интерфейса.

При нажатии на кнопку добавления появляется окно с полями для ввода данных и кнопкой "add", при нажатии на которую данные добавляются в таблицу

Для обновления данных необходимо нажать на синюю кнопку в строке, которую вы желаете изменить.
После чего запись перейдет в режим редактирования. При повторном нажатии на кнопку изменения будут внесены.

Для удаления записи нажмите на красную кнопку в строке таблице. Появится окно подтверждения удаления, при нажатии на кнопку ok запись удалится.

Выход их системы

Для выхода их системы нажмите на поле "Log out" в панели навигации
